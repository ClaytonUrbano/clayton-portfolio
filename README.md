# Portfólio Profissional

Este repositório contém o código do Portfólio Profissional desenvolvido como parte do Projeto 1 – Portfólio Profissional Interativo. O objetivo do projeto é construir uma presença digital pública, clara e organizada, servindo como vitrine profissional para apresentação pessoal e de projetos.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Git
- GitHub Pages

## Estrutura do Portfólio

O site é composto por seções bem definidas, com foco em clareza e semântica:

- Apresentação pessoal
- Formação acadêmica
- Vitrine de projetos
- Integração com GitHub e LinkedIn
- Canal de contato
- Layout responsivo

## Justificativa Técnica

O portfólio foi desenvolvido como um website estático utilizando exclusivamente HTML5 e CSS3, conforme proposto no projeto. As escolhas técnicas priorizaram estrutura semântica, organização visual e facilidade de leitura, garantindo uma apresentação clara das informações.

O versionamento do projeto foi realizado com Git, utilizando commits progressivos para registrar ajustes estruturais, refinamento de conteúdo, melhorias visuais e consolidação da entrega final. Essa estratégia evidencia o uso consciente da ferramenta de controle de versão e a evolução incremental do projeto até sua versão publicada.

A publicação foi realizada por meio do GitHub Pages, garantindo acesso público, rápido e funcional ao portfólio, atendendo aos critérios de deploy e visibilidade profissional.
